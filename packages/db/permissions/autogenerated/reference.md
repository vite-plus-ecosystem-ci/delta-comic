## Default Permission

Enables Delta Comic database file-store commands.

#### Granted Permissions

Allows the frontend database package to persist native store values
to the application data directory.

#### This default permission set includes the following:

- `allow-native-store-get`
- `allow-native-store-remove`
- `allow-native-store-set`
