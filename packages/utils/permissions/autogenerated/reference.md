## Default Permission

Enables Delta Comic WebView utility commands.

#### Granted Permissions

Allows the app to open temporary authentication pages, inject auth helper code,
read WebView cookie/localStorage/sessionStorage snapshots, and close those pages.

#### This default permission set includes the following:

- `allow-webview-auth-data`
- `allow-webview-auth-data-all`
- `allow-webview-auth-data-current`
- `allow-webview-close-current-page`
- `allow-webview-close-page`
- `allow-webview-iframe-auth-data`
- `allow-webview-inject-code`
- `allow-webview-open-page`
